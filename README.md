# Sand Timer (TUI)

`Reliable Sand Timer` は、Rustで書かれたターミナルユーザーインターフェース (TUI) ベースの砂時計タイマーです。物理演算に基づいて砂の落下をシミュレートし、指定された時間で砂が落ち切るように動的に調整されます。さらに、巨大なデジタル時計も備え、タイマーと現在時刻を同時に確認できます。

## ✨ 特徴

*   **リアルタイム砂物理シミュレーション**:
    *   セルオートマトン方式による砂の忠実な動き。
    *   壁摩擦、砂摩擦の調整可能。
    *   設定時間に合わせて砂の落下速度（ゲート幅）を自動調整。
*   **多機能タイマー**:
    *   時間設定、実行、一時停止、終了の4つの状態。
    *   経過時間の正確な表示。
*   **デジタル時計**:
    *   砂時計の横に、大きく視認性の高いブロック文字で現在時刻を表示。
    *   `t` キーで12時間表示と24時間表示を切り替え可能。
*   **インタラクティブな操作**:
    *   時・分・秒の直感的な設定。
    *   摩擦係数の調整。
    *   一時停止、再開、リセット、終了操作。
*   **設定の永続化**: 最後に使用したタイマー設定、摩擦係数、時計表示形式は `~/.sand_timer.toml` に自動保存・読み込みされます。
*   **CachyOS/Archフレンドリー**: パフォーマンスとRustエコシステムの標準を意識した実装。

## 🚀 インストール

### 前提条件

*   Rustプログラミング言語 (最新の安定版を推奨)
*   Cargo (Rustのパッケージマネージャー、Rustインストール時に含まれます)

### ビルドと実行

1.  **リポジトリをクローン**:
    ```bash
    git clone https://github.com/Miserry/sand_timer.git
    cd sand_timer
    ```
    (↑ `your-username` はあなたのGitHubユーザー名に置き換えてください)

2.  **プロジェクトをビルドして実行**:
    ```bash
    cargo run
    ```
    または、リリースビルドを作成して実行することも可能です (最適化により高速になります):
    ```bash
    cargo build --release
    ./target/release/sand_timer
    ```

## 🎮 使用方法

アプリケーションはターミナル上で動作します。

### キーバインド

| キー       | アクション                               | 説明                                                       |
| :--------- | :--------------------------------------- | :--------------------------------------------------------- |
| `←` / `→`  | タイマー設定単位の選択 (時/分/秒)        | `SETTING` モード時のみ。                                   |
| `↑` / `↓`  | 選択中のタイマー単位の値調整             | `SETTING` モード時のみ。                                   |
| `数字キー` | 選択中のタイマー単位に直接入力           | `SETTING` モード時のみ。                                   |
| `Space`    | タイマーの開始 / 一時停止 / 再開         | `SETTING` から `RUNNING` へ、`PAUSED` から `RUNNING` へ。  |
| `r`        | タイマーのリセット                       | 現在のタイマー設定で砂時計を初期状態に戻します。           |
| `[` / `]`  | 壁摩擦 (`Wall Friction`) の調整          | 物理シミュレーションにおける壁と砂の摩擦を増減させます。   |
| `{` / `}`  | 砂摩擦 (`Sand Friction`) の調整          | 物理シミュレーションにおける砂同士の摩擦を増減させます。   |
| `t`        | デジタル時計の表示形式切り替え           | 12時間表示 (AM/PM) と24時間表示を切り替えます。            |
| `q`        | アプリケーションの終了                   | 現在の設定を保存してアプリケーションを終了します。         |

## ⚙️ 設定

設定は、以下のパスにある `~/.sand_timer.toml` ファイルに自動的に保存されます。
(例: `/home/youruser/.sand_timer.toml`)

このファイルを直接編集することで、起動時の初期設定を変更することも可能です。

```toml
# ~/.sand_timer.toml の例
hour = 0
minute = 0
second = 30
wall_friction = 0.1
sand_friction = 0.02
is_24h_format = true
```

*   `hour`, `minute`, `second`: タイマーの初期設定時間 (時、分、秒)。
*   `wall_friction`: 砂と壁の摩擦係数 (0.0 から 1.0 の間)。
*   `sand_friction`: 砂同士の摩擦係数 (0.0 から 1.0 の間)。
*   `is_24h_format`: デジタル時計の表示形式 (24時間表示なら `true`、12時間表示なら `false`)。
